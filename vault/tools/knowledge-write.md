# knowledge_write

**Type:** Tool
**Category:** See system_map

## Description

Write or append to a knowledge topic. Use for recipes, gotchas, patterns learned from experience.

## Parameters

- `topic` (string): Topic name (alphanumeric, hyphens, underscores)
- `content` (string): Content to write (markdown)
- `mode` (string): Write mode: 'overwrite' (default) or 'append'
