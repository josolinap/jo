# send_photo

**Type:** Tool
**Category:** See system_map

## Description

Send a base64-encoded image (PNG) to the owner's Telegram chat. Use after browse_page(output='screenshot') or browser_action(action='screenshot'). Pass the base64 string from the screenshot result as image_base64.

## Parameters

- `image_base64` (string): Base64-encoded PNG image data
- `caption` (string): Optional caption for the photo

Called automatically when needed. Use `system_map` tool to see full tool list.
