# system_map

**Type:** Tool
**Category:** See system_map

## Description

Get a dynamic map of the Jo system showing: all available tools (from registry), tool categories, drive paths and their status, API keys, external dependencies. Queries the system in real-time - always accurate. Use format='json' for structured data.

## Parameters

- `format` (string): Output format: 'text' for human-readable, 'json' for programmatic use

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
