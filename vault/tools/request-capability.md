# request_capability

**Type:** Tool
**Category:** See system_map

## Description

Request a new capability or resource. Use when you encounter a limitation and need something to overcome it. Requests are logged for review and fulfillment.

## Parameters

- `capability` (string): What is needed (e.g., 'GitHub API access', 'database connection')
- `reason` (string): Why it's needed
- `priority` (string): No description

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
