# find_definitions

**Type:** Tool
**Category:** See system_map

## Description

Find where a function or class is defined in the codebase. Returns file path and line number of the definition.

## Parameters

- `function_name` (string): Name of the function or class to find

Called automatically when needed. Use `system_map` tool to see full tool list.
