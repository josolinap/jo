# record_tool_combination

**Type:** Tool
**Category:** See system_map

## Description

Manually record that a set of tools work well together for a specific task type.

## Parameters

- `tools` (array): List of tool names that work well together
- `description` (string): Description of what they were used for
- `task_type` (string): Task type context (code, research, vault, git, web, system)

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
