# recall_lessons

**Type:** Tool
**Category:** See system_map

## Description

Recall lessons learned about a topic or pattern. Use this when facing a familiar problem or before making a risky decision. Jo should search lessons before attempting unfamiliar tasks.

## Parameters

- `topic` (string): Optional topic/tags to filter by (e.g., 'git', 'async')
- `limit` (integer): Maximum number of lessons to return (default: 10)

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
