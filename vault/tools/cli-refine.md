# cli_refine

**Type:** Tool
**Category:** See system_map

## Description

Expand coverage of an existing CLI harness. Performs gap analysis between full software capabilities and current CLI, then implements new commands, tests, and documentation. Incremental and non-destructive.

## Parameters

- `software_path` (string): Path to existing CLI harness
- `focus` (string): Specific functionality area to expand (e.g., 'batch processing')

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
