# research_pipeline

**Type:** Tool
**Category:** See system_map

## Description

Run systematic research: search -> fetch top results -> cross-verify -> synthesize. This is the primary research tool for thorough investigation of topics. Returns structured findings with source citations and confidence scores.

## Parameters

- `query` (string): Research question/topic
- `num_sources` (integer): Number of sources to analyze (default 5)
- `date_filter` (string): Filter by recency (default: week)
