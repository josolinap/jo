# list_github_issues

**Type:** Tool
**Category:** See system_map

## Description

List GitHub issues. Use to check for new tasks, bug reports, or feature requests from the creator or contributors.

## Parameters

- `state` (string): Filter by state
- `labels` (string): Filter by label (comma-separated)
- `limit` (integer): Max issues to return (max 50)
