# find_connections

**Type:** Tool
**Category:** See system_map

## Description

Find connections between two concepts, files, or ideas. Shows the path between them and what's related to each.

## Parameters

- `concept_a` (string): First concept or file name
- `concept_b` (string): Second concept or file name
