# browser_profile_save

**Type:** Tool
**Category:** See system_map

## Description

Save current browser state (cookies, localStorage) to a named profile. This allows persistent sessions - login once, reuse later. Similar to PinchTab's profile management.

## Parameters

- `name` (string): Profile name
- `description` (string): Optional description
