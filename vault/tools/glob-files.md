# glob_files

**Type:** Tool
**Category:** See system_map

## Description

Find files matching a glob pattern (e.g., '**/*.py'). Searches from repo root unless path specified.

## Parameters

- `pattern` (string): Glob pattern (e.g., '**/*.py', 'tests/*.py')
- `path` (string): Optional directory to search in

Called automatically when needed. Use `system_map` tool to see full tool list.
