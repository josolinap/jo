# vault_list

**Type:** Tool
**Category:** See system_map

## Description

List all vault notes, optionally filtered by folder or tags.

## Parameters

- `folder` (string): Filter by folder name
- `tags` (string): Filter by comma-separated tags
- `status` (string): Filter by status: active, draft, archived
