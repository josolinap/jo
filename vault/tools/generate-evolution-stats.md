# generate_evolution_stats

**Type:** Tool
**Category:** See system_map

## Description

Generate Evolution Time-Lapse data from git history and push to the webapp dashboard. Collects per-commit metrics across three axes: Technical (Python lines of code), Philosophical (BIBLE.md size), Self-Concept (SYSTEM.md size). Pushes docs/evolution.json via GitHub API. Safe to call anytime; takes 15-30s for full history scan.
