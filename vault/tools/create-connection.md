# create_connection

**Type:** Tool
**Category:** See system_map

## Description

Create a new connection between concepts by adding wikilinks. Links two ideas/files in the vault.

## Parameters

- `from_concept` (string): Source concept or note name
- `to_concept` (string): Target concept or note name
- `connection_type` (string): Type of connection (default: related)

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
