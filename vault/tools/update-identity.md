# update_identity

**Type:** Tool
**Category:** See system_map

## Description

Update your identity manifest (who you are, who you want to become). Saves to both memory/identity.md and vault/concepts/identity.md. Use commit=True only if you want this in GitHub (triggers restart). For runtime changes, commit=False (no restart needed).

## Parameters

- `content` (string): Full identity content
- `commit` (boolean): Commit to GitHub? (default: false, no restart needed)

Called automatically when needed. Use `system_map` tool to see full tool list.
