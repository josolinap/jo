# web_search

**Type:** Tool
**Category:** See system_map

## Description

Search the web for information. Returns structured results with URLs, titles, and snippets. Use for initial research and finding relevant sources. Supports date filtering: today, week, month, year.

## Parameters

- `query` (string): Search query
- `num_results` (integer): Number of results (default 10)
- `date_filter` (string): Filter by date: today, week, month, year (default: all)
