# wait_for_task

**Type:** Tool
**Category:** See system_map

## Description

Check if a subtask has completed. Returns result if done, or 'still running' message. Call repeatedly to poll. Default timeout: 120s.

## Parameters

- `task_id` (string): Task ID to check

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
