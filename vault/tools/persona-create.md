# persona_create

**Type:** Tool
**Category:** See system_map

## Description

Create a new agent persona with specific characteristics (inspired by MiroFish).

## Parameters

- `name` (string): Name of the persona
- `description` (string): What this persona does
- `traits` (string): JSON of personality traits (e.g., {"creative": 8})
- `expertise` (string): JSON of expertise areas (e.g., {"python": 9})

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
