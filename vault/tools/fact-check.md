# fact_check

**Type:** Tool
**Category:** See system_map

## Description

Verify a claim against multiple sources. Searches for corroborating or contradicting evidence. Returns verification status: confirmed, disputed, unverified.

## Parameters

- `claim` (string): Claim to verify
- `sources` (string): URLs or sources to check against

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
