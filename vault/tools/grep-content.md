# grep_content

**Type:** Tool
**Category:** See system_map

## Description

Search for a regex pattern within files. Useful for finding code patterns across the codebase.

## Parameters

- `pattern` (string): Regex pattern to search for
- `path` (string): Optional directory to search in
- `file_pattern` (string): File pattern to match (e.g., '*.py', '*.md')
- `context_lines` (integer): Number of context lines
