# send_owner_message

**Type:** Tool
**Category:** See system_map

## Description

Send a proactive message to the owner. Use when you have something genuinely worth saying — an insight, a question, or an invitation to collaborate. This is NOT for task responses (those go automatically).

## Parameters

- `text` (string): Message text
- `reason` (string): Why you're reaching out (logged, not sent)

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
