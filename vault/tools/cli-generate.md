# cli_generate

**Type:** Tool
**Category:** See system_map

## Description

Generate a complete CLI harness for any software. Based on CLI-Anything 7-phase pipeline: Analyze → Design → Implement → Plan Tests → Write Tests → Document → Publish. Takes a software path or GitHub repo and produces an installable Click CLI.

## Parameters

- `software_path` (string): Path to software directory or GitHub repo URL
- `focus` (string): Optional focus area (e.g., 'image processing', 'export')
- `json_output` (boolean): Return results in JSON format for agent consumption

Called automatically when needed. Use `system_map` tool to see full tool list.
