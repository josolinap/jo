# switch_model

**Type:** Tool
**Category:** See system_map

## Description

Switch to a different LLM model or reasoning effort level. Use when you need more power (complex code, deep reasoning) or want to save budget (simple tasks). Takes effect on next round.

## Parameters

- `model` (string): Model name (e.g. anthropic/claude-sonnet-4). Leave empty to keep current.
- `effort` (string): Reasoning effort level. Leave empty to keep current.

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
