# vault_create

**Type:** Tool
**Category:** See system_map

## Description

Create a new note in Jo's knowledge vault. Use to capture concepts, decisions, or ideas with proper structure.

## Parameters

- `title` (string): Note title
- `folder` (string): Folder: concepts, projects, tools, journal (default: concepts)
- `content` (string): Initial content (optional)
- `tags` (string): Comma-separated tags (optional)
- `type` (string): Note type: reference, project, concept, tool (default: reference)
- `status` (string): Status: active, draft, archived (default: active)

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
