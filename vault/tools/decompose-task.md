# decompose_task

**Type:** Tool
**Category:** See system_map

## Description

Break down a complex task into focused subtasks with assigned roles. Use this BEFORE delegating to understand the task structure. Returns subtask list with roles and dependencies.

## Parameters

- `task` (string): Complex task to decompose
- `context` (string): Optional background context

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
