# activate_skill

**Type:** Tool
**Category:** See system_map

## Description

Activate a specialized cognitive mode (skill). Available skills: plan (founder mode), plan-eng (engineering), review (paranoid code review), ship (release), qa (testing), retro (team retro). Each skill changes how Jo approaches the task.

## Parameters

- `skill` (string): Skill to activate (e.g., 'plan', 'review', 'ship', 'qa', 'retro')
- `task` (string): The task or context to apply this skill to

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
