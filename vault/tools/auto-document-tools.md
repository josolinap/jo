# auto_document_tools

**Type:** Tool
**Category:** See system_map

## Description

Automatically find and document orphan tools (tools without vault entries). Scans all 130+ registered tools and creates vault/tools/ entries. Run without create_entries=true to preview, or with create_entries=true to generate.

## Parameters

- `create_entries` (boolean): If true, create vault entries for orphan tools

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
