# validate_connection

**Type:** Tool
**Category:** See system_map

## Description

Verify a connection exists between two concepts. Returns evidence for the connection and checks if it's still valid. Helps reduce hallucinations by confirming knowledge links.

## Parameters

- `source` (string): Source concept or file
- `target` (string): Target concept or file
