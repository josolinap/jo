# find_callers

**Type:** Tool
**Category:** See system_map

## Description

Find all places in the codebase that call a specific function. Use before editing a function to know what might break. Returns file paths and line numbers.

## Parameters

- `function_name` (string): Name of the function to search for

## Usage

Called automatically when needed. Use `system_map` tool to see full tool list.
