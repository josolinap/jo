# chat_history

**Type:** Tool
**Category:** See system_map

## Description

Retrieve messages from chat history. Supports search.

## Parameters

- `count` (integer): Number of messages (from latest)
- `offset` (integer): Skip N from end (pagination)
- `search` (string): Text filter
