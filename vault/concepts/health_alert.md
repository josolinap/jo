# Health Alert

System alerts generated by health monitoring.

## Alert Types
- **Identity staleness** - identity.md not updated within 8 hours
- **Module size violation** - Python module exceeds 1000 lines
- **Drift violation** - Configuration or state drift detected
- **Test failure** - Test suite has failing tests
- **Syntax error** - Python syntax check fails
- **Consecutive failures** - Evolution cycle failure streak
