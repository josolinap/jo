# Multi-Agent Architecture

Jo can delegate work to sub-agents for parallel execution.

## Mechanisms
- [[delegated_reasoning]] - Spawning sub-agents for complex tasks
- [[agent_coordinator]] - Managing parallel agent execution
- [[schedule_task]] - Queuing work for later execution

## When to Use
- Task decomposes naturally into independent subtasks
- Single-agent context limits are exceeded
- Specialized agents improve quality
- Parallel execution saves time
