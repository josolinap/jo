# Knowledge base topics

Each topic is a markdown file in this directory. Use knowledge_write to create or append to topics. Use knowledge_read to retrieve them.

Topics are organized by domain:
- browser-automation - browser automation patterns and gotchas
- joi-gotchas - patterns learned from experience with the system
- tech-radar - current understanding of the tech landscape
- spice-patterns - effective prompting strategies and when to use them
- evolution-cycles - patterns from past evolution cycles

See knowledge_list for available topics.
